\# Lesson 09: Building Image Generation Applications



\## Microsoft Course Concept



This lesson introduces image generation applications. It explains that generative AI can create images from text descriptions and discusses models such as DALL-E and Midjourney.



\## My Adaptation



Because direct image-generation APIs may require paid access, I first built a Research Image Prompt Builder. The app generates structured, safe, professional prompts that can be used with image generation models.



\## Key Concepts Practised



\- Image generation prompts

\- Meta prompts

\- Safety boundaries

\- Audience-aware visual design

\- Professional prompt structure

\- Research communication visuals



\## Build Completed



Research Image Prompt Builder.



\## Main Lesson



A strong image generation application does not only pass user text directly to an image model. It should shape the request using meta prompts, style constraints, audience needs, and safety boundaries.



\## Build 2: Prompt Builder + Save Prompt to File



I upgraded the Research Image Prompt Builder so that it now saves each generated image prompt to a text file. This makes it possible to build a reusable prompt library for future LinkedIn graphics, grant visuals, scientific figure concepts, and other research communication assets.



\## Build 4: Thumbnail Prompt Specialist



I built a specialized thumbnail prompt generator for LinkedIn, FRIN, newsletter, academic, and professional communication visuals.



This app uses meta prompts to define safety, style, audience, platform, brand identity, colour palette, mood, and visual hierarchy before producing a detailed image generation prompt.



This extends the Lesson 9 concept of image generation into a practical research communication and personal branding workflow.

\## Lesson 09 Summary



This lesson demonstrated that image generation applications require more than simple text-to-image prompting. A useful application should guide the user through image purpose, target audience, visual style, safety boundaries, orientation, and output requirements.



My adaptation focused on building reusable prompt-generation tools for academic, research, and professional communication.
